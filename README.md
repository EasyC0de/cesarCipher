# CC

## Getting Started

Just a simple Cesar Cipher program in C.

### Usage
USAGE: CC [OPTION] [SOURCE] [KEY]
Example: CC -e file.txt 5

 - -e: Encrypt file
 - -d: Decrypt file
 - -h: Show help
